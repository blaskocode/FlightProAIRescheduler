# Demo Video Script: Flight Schedule Pro AI Rescheduler

**Target Length**: 5-10 minutes  
**Format**: Screen recording with narration

---

## Introduction (30 seconds)

**Visual**: Show landing page, then dashboard

**Narration**:
"Welcome to Flight Schedule Pro AI Rescheduler. This system automatically monitors weather conditions for flight lessons and uses AI to suggest optimal rescheduling options when weather conflicts are detected. Let me show you how it works."

---

## Setup & Dashboard Overview (1 minute)

**Visual**: 
- Navigate to dashboard
- Show flight list with upcoming flights
- Point out weather alerts section

**Narration**:
"Here's the main dashboard. You can see all upcoming flights, and the system automatically checks weather conditions hourly. When a weather conflict is detected, it appears here in the Weather Alerts section. Let me show you what happens when a conflict is detected."

---

## Weather Conflict Detection (1.5 minutes)

**Visual**:
- Show a flight scheduled for tomorrow
- Navigate to admin settings
- Click "Run Manual Weather Check"
- Show processing status
- Return to dashboard
- Show weather alert appearing

**Narration**:
"The system continuously monitors weather. When a conflict is detected - for example, visibility below minimums for a student pilot - an alert is automatically generated. Notice how the system considers the student's training level - a student pilot needs much better weather than an instrument-rated pilot."

**Key Points to Highlight**:
- Training-level specific minimums
- Automatic hourly checks
- Real-time alerts

---

## AI Rescheduling (2 minutes)

**Visual**:
- Click on flight with weather alert
- Click "Request Reschedule" button
- Show modal with 3 AI-generated suggestions
- Walk through each suggestion, highlighting:
  - Suggested date/time
  - Instructor assignment
  - Aircraft assignment
  - Reasoning for each option
  - Weather forecast for suggested time

**Narration**:
"When a weather conflict is detected, the AI analyzes multiple factors to generate three optimal reschedule options. It considers student availability, instructor schedules, aircraft availability, and weather forecasts for the suggested times. Each option includes detailed reasoning - for example, this option maintains instructor continuity, which is important for student progress."

**Key Points to Highlight**:
- 3 ranked suggestions
- AI reasoning for each
- Considers multiple factors
- Weather forecast included

---

## Student Confirmation (1 minute)

**Visual**:
- Select one of the reschedule options (radio button)
- Click "Select Option"
- Show confirmation message
- Show flight status changing to "Reschedule Pending"

**Narration**:
"The student can easily select their preferred option. Once selected, the system creates a reschedule request and notifies the instructor. The original flight is marked as 'Reschedule Pending' while waiting for instructor confirmation."

**Key Points to Highlight**:
- Simple one-click selection
- Automatic notification
- Status tracking

---

## Instructor Confirmation (1 minute)

**Visual**:
- Switch to instructor account (or show instructor view)
- Show flight with "Reschedule Pending" status
- Click "Confirm Reschedule" button
- Show confirmation
- Show new flight created
- Show original flight status updated

**Narration**:
"The instructor receives a notification and can review the student's selected option. With one click, they confirm the reschedule. The system automatically creates a new flight booking and updates the original flight status. Both parties are notified immediately."

**Key Points to Highlight**:
- Two-step confirmation process
- Automatic booking creation
- Real-time notifications

---

## Additional Features (1.5 minutes)

**Visual**:
- Show student progress tracking
- Show aircraft squawk reporting
- Show admin analytics dashboard
- Show weather analytics

**Narration**:
"The system includes many additional features. Students can track their training progress through the syllabus. Instructors can report aircraft squawks, which automatically cancel affected flights and trigger reschedule suggestions. Admins have access to comprehensive analytics showing weather impact, resource utilization, and student progress."

**Key Points to Highlight**:
- Progress tracking
- Squawk system
- Analytics dashboard
- Multi-role support

---

## Metrics Dashboard (1 minute)

**Visual**:
- Navigate to admin dashboard
- Show metrics:
  - Weather cancellations
  - Successful reschedules
  - Average rescheduling time
  - Revenue protected
- Show date range filters

**Narration**:
"Admins can view detailed metrics showing the system's impact. Here you can see weather cancellations, successful reschedules, and average rescheduling time - which is a key metric showing how quickly we get students back in the air. The system helps protect revenue by maximizing reschedule success rates."

**Key Points to Highlight**:
- Comprehensive metrics
- Revenue protection
- Performance tracking

---

## Conclusion (30 seconds)

**Visual**: Return to dashboard, show summary

**Narration**:
"Flight Schedule Pro AI Rescheduler automates weather conflict detection and rescheduling, reducing training delays and protecting revenue. The AI-powered suggestions consider multiple factors to find optimal reschedule options, and the two-step confirmation process ensures all parties are aligned. Thank you for watching!"

## Tips for Recording

1. **Use a script**: Read from this script to stay on track
2. **Practice first**: Do a dry run to check timing
3. **Show, don't tell**: Let the UI speak for itself
4. **Highlight key features**: Pause to emphasize important points
5. **Keep it concise**: Aim for 7-8 minutes total
6. **Edit if needed**: Cut out pauses and mistakes
7. **Add captions**: Consider adding text overlays for key metrics
